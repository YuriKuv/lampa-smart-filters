# Smart Filters Plugin for Lampa

Расширение для Lampa, позволяющее сохранять настраиваемые фильтры с поддержкой категорий.

## Возможности

- 📁 Создание категорий для фильтров
- 🔖 Сохранение любых параметров фильтрации
- ⚙️ Интеграция в настройки Lampa
- 💾 Автосохранение последнего фильтра
- 🎨 Полная адаптация под интерфейс Lampa

## Установка

1. Скопируйте ссылку: `https://YuriKuv.github.io/lampa-smart-filters/main.js`
2. В Lampa: Настройки → Расширения → Добавить плагин
3. Вставьте ссылку и подтвердите

## Требования

- Lampa 3.0 или выше
- Доступ в интернет
