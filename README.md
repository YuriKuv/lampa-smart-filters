# Закладки для Lampa (Bookmarks for Lampa)

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Lampa](https://img.shields.io/badge/Lampa-1.0%2B-green.svg)](https://lampa.mx)

Плагин для приложения Lampa, позволяющий сохранять любые разделы, подборки и фильтры в виде закладок для быстрого доступа через левое меню.

## 📌 Возможности

- ✅ **Сохранение любых разделов** — фильмы, сериалы, мультфильмы, аниме, каталог, персоны и т.д.
- ✅ **Сохранение подразделов** — "Сейчас смотрят", "Популярное", "Топ рейтинга" и другие
- ✅ **Сохранение фильтров** — все применённые фильтры (жанры, годы, язык, рейтинг) сохраняются
- ✅ **Быстрый доступ** — все закладки отображаются в левом меню
- ✅ **Удаление закладок** — долгое нажатие на закладке вызывает диалог удаления
- ✅ **Удаление всех закладок** — отдельная кнопка для массовой очистки
- ✅ **Гибкое расположение кнопок** — кнопки можно разместить в левом меню или в верхней панели
- ✅ **Кроссплатформенность** — работает на всех платформах Lampa

## 📥 Установка

### Способ 1: Через настройки Lampa

1. Откройте `Настройки → Расширения → Добавить плагин`
2. Вставьте ссылку на файл плагина: https://yurikuv.github.io/your-bookmarks/yba.js
3. Нажмите "Добавить"

### Способ 2: Локальный файл

1. Скачайте файл `yba.js`
2. Поместите его в папку `plugins` вашего сервера с Lampa
3. Добавьте ссылку на плагин в настройках

## 🚀 Использование

### Создание закладки

1. Откройте нужный раздел (Фильмы, Сериалы, Мультфильмы и т.д.) перейдите в нужный подраздел с помощью кнопки "Ещё" или примените фильтр (жанры, годы, язык) и нажмите начать поиск
2. Нажмите кнопку **"Сохранить закладку"** в левом меню или верхней панели
3. Введите название закладки
4. Закладка появится в разделе закладок левого меню

### Открытие закладки

- Просто нажмите на название закладки в левом меню

### Удаление закладки

- **Долгое нажатие** (удержание ~0.8 сек) на названии закладки
- Подтвердите удаление в появившемся диалоге

### Удаление всех закладок

- Нажмите кнопку **"Удалить все закладки"** в левом меню или верхней панели
- Подтвердите удаление

## ⚙️ Настройки

Настройки находятся в разделе: `Настройки → Интерфейс`

| Параметр | Описание |
|----------|----------|
| **Кнопка "Сохранить закладку"** | Выбор расположения: в левом меню (над Настройками) или в верхней панели |
| **Кнопка "Удалить все закладки"** | Выбор расположения: в левом меню (над Настройками) или в верхней панели |

## 📝 Примеры использования

| Действие | Результат |
|----------|-----------|
| Сохранение подборки "Боевики 2024" | Примените фильтр "Боевики" и "2024", сохраните закладку |
| Сохранение раздела "Сейчас смотрят" | Откройте раздел, нажмите "Сохранить закладку" |
| Сохранение русских фильмов с рейтингом выше 7 | Настройте фильтры, сохраните закладку |
| Сохранение любимого жанра в каталоге | Откройте нужный жанр в каталоге, сохраните закладку |

## 🔧 Требования

- Lampa версии 1.0.0 и выше
- Поддержка JavaScript ES5+
