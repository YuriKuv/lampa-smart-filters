# Закладки+ для Lampa (Bookmarks for Lampa)

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Lampa](https://img.shields.io/badge/Lampa-1.0%2B-green.svg)](https://lampa.mx)

📌 Закладки+ — Плагин для Lampa

Плагин Закладки+ добавляет в Lampa удобную систему создания закладок. Вы можете сохранять любые страницы, подборки и результаты фильтрации в один клик — и мгновенно открывать их потом из меню.
🔧 Возможности
Функция	Описание
✅ Сохранение страниц	Сохраните любой раздел, подборку, результат фильтрации или поиска
✅ Быстрый доступ	Все закладки отображаются в левом меню — одна строка, одно нажатие
✅ Удаление закладок	Долгое нажатие на пульте / удержание мыши — и закладка удаляется
✅ Гибкое расположение кнопки	Кнопка «Добавить закладку» может быть в боковом меню или в верхней панели
✅ Очистка всех закладок	Пункт в настройках для массового удаления
✅ Кроссплатформенность	Полностью работает на Android TV, WebOS, Tizen, Windows, macOS, браузере
✅ Нет лишних диалогов	Используется только нативный Lampa.Input.edit — корректно работает с пультом и клавиатурой
📥 Установка
Способ 1 — через интерфейс Lampa

    Откройте Настройки → Расширения → Добавить плагин

    Вставьте ссылку на плагин:
    https://yurikuv.github.io/your-bookmarks/yba.js

    Нажмите «Добавить»

    Перезагрузите Lampa

Способ 2 — локальный файл

    Скачайте файл yba.js

    Поместите его на свой сервер или в локальную папку

    Добавьте ссылку в настройках Lampa

🧭 Использование
1️⃣ Создание закладки

    Откройте нужный раздел (например, «Боевики 2024» или «Популярное»)

    Нажмите кнопку «Добавить закладку»:

        в боковом меню (если кнопка там)

        или в верхней панели (иконка ➕)

    Введите название закладки в появившемся окне

    Закладка появится в основном меню

    🧠 Плагин автоматически определяет, можно ли сохранить текущую страницу (корневые разделы вроде «Фильмы» или «Сериалы» сохранить нельзя).

2️⃣ Открытие закладки

    Перейдите в левое меню

    Нажмите на любую закладку из списка

    Страница откроется мгновенно

3️⃣ Удаление закладки

    Нажмите и удерживайте закладку (долгое нажатие)

    Подтвердите удаление в диалоговом окне

4️⃣ Удаление всех закладок

    Перейдите в Настройки → Закладки+

    Нажмите «Очистить все закладки»

    Подтвердите действие

❓ Часто задаваемые вопросы
Почему нельзя сохранить «Фильмы» или «Сериалы»?

Плагин не даёт сохранять корневые разделы, потому что они уже есть в меню. Сохранять имеет смысл только подборки, фильтры и конкретные страницы.
Закладки синхронизируются между устройствами?

Плагин не использует внешние серверы. Закладки хранятся локально на каждом устройстве.
Если нужна синхронизация — используйте CUB или плагин Profiles.
Кнопка не появляется после установки?

Проверьте:

    Плагин активен в списке расширений

    Вы перезагрузили Lampa

    В настройках выбран правильный режим отображения кнопки
